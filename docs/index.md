# binomcikit

Inference on a single binomial proportion — a Python port of the R package
[`proportion`](https://github.com/RajeswaranV/proportion).

`binomcikit` provides confidence intervals (frequentist, adjusted,
continuity-corrected, exact and Bayesian) and the machinery to evaluate them:
coverage probability, expected length, p-confidence / p-bias, error & failure,
and a suite of Bayesian tools.
