# API reference

Complete per-function documentation, grouped by family. Each page gives the signature, what the function does, the maths, an example, and how it maps to the original R function.

```{tip}
Function names follow a strict scheme — see the {doc}`naming convention <../getting_started>`. Add the `plot` prefix to any numeric function to get its plotting counterpart (see the {doc}`gallery <../gallery>`).
```
